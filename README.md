Persona Juru Masak menggunakan Gemini AI

Berikut langkah-langkah ntuk menjalankan program
1. Download Python (https://www.python.org/downloads/)
2. Install Python
3. Buka project nya
4. Install Google GenAI menggunakan terminal dengan perintah "python3 -m pip install google-genai"
5. Install env menggunakan terminal dengan perintah "python3 -m pip install python-dotenv"
6. Get APIKey Gemini (https://aistudio.google.com/api-keys)
7. Buat file .env di folder projectnya lalu paste GEMINI_API_KEY=APIKey nya
8. Jalankan programnya menggunakan terminal dengan perintah "python3 hello.py"
